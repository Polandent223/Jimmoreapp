INVERSIONES JIMMORE - FASE 13 PRO

Incluye:
1. Respaldo interno y descarga de respaldo JSON.
2. Restaurar respaldo JSON.
3. Plantilla CSV para importar productos desde Excel.
4. Importación masiva de productos.
5. Exportación de productos, clientes y facturas en CSV.
6. Factura térmica 80mm para impresora de punto de venta.
7. Etiquetas de productos con código visual, precio USD/Bs y ubicación.
8. Escáner de código con cámara del celular cuando el navegador lo permita.
9. Reporte de stock bajo.
10. Valoración de inventario por costo y precio de venta.

IMPORTANTE:
- Para usar en varias computadoras/celulares al mismo tiempo, conecta Firebase en Configuración.
- Para publicar gratis, sube la carpeta completa a GitHub Pages.
- La factura térmica para cliente no muestra costo, ganancia ni control interno.
