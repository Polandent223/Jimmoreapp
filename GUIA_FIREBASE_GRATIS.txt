INVERSIONES JIMMORE 0331 C.A. - FASE 2 FIREBASE GRATIS

OBJETIVO
Esta fase permite que el sistema funcione en varias PC, laptop, tablet y celular al mismo tiempo usando Firebase Realtime Database gratis.

PASOS EN FIREBASE
1. Entra a Firebase Console.
2. Crea un proyecto nuevo.
3. En Realtime Database, crea una base de datos.
4. Para comenzar en prueba, usa estas reglas temporales:

{
  "rules": {
    ".read": true,
    ".write": true
  }
}

IMPORTANTE: Estas reglas son para prueba. Después se pueden cerrar con usuarios y permisos.

5. En configuración del proyecto, crea una app web.
6. Copia el bloque firebaseConfig completo.
7. Abre el sistema Jimmore.
8. Entra como Administrador > Configuración.
9. Pega el firebaseConfig.
10. Presiona Guardar configuración.
11. Presiona Conectar Firebase.
12. Presiona Subir datos locales a Firebase si ya tenías productos o facturas.

QUÉ AGREGA ESTA FASE
- Sincronización en tiempo real con Firebase.
- Botón de conectar Firebase.
- Botón para subir respaldo local a Firebase.
- Botón para bajar datos desde Firebase.
- Tasa BCV manual y botón automático por internet.
- El sistema sigue funcionando sin internet usando localStorage.

PRÓXIMA FASE RECOMENDADA
Fase 3: usuarios reales con correo/contraseña y permisos por rol para administrador, vendedor, almacén, mecánico/taller/pintor y cliente.
