INVERSIONES JIMMORE - FASE 9 PRO

Incluye:
- Factura para cliente sin control interno.
- Factura interna solo para administración con costo, ganancia y auditoría.
- Botón "Imprimir para cliente".
- Botón "Ver factura interna".
- Control de caja separado por Efectivo USD, Efectivo Bs, Pago móvil, Zelle, Binance, Tarjeta/POS y Cuentas por cobrar.
- Seguimiento del cliente por código de trabajo.

Acceso de prueba:
Usuario: admin
Clave: 1234

Para usar:
1. Abre index.html.
2. Entra como admin.
3. Crea productos/clientes.
4. Haz una venta en Punto de Venta.
5. En Facturación usa "Imprimir para cliente" para entregar al cliente.
6. Usa "Ver factura interna" solo para administración.
