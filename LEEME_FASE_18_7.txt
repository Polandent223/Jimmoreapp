FASE 18.7 PRO - SEGURIDAD Y RESPALDO

Mejoras agregadas:
- Botón Crear respaldo completo en Configuración.
- Importar respaldo JSON con clave de administrador.
- Restaurar sistema para entrega con doble confirmación y clave de administrador.
- Limpieza de ventas, facturas, clientes, vehículos, trabajos, evidencias, cuentas por cobrar/pagar, caja, compras y notificaciones.
- Opciones para mantener productos, empresa/BCV/Firebase y usuarios/permisos.
- Auditoría del proceso de respaldo/restauración.

Recomendación antes de entregar:
1. Entrar como Administrador.
2. Ir a Configuración.
3. Crear respaldo completo.
4. Usar Restaurar sistema para entrega.
5. Revisar que productos, usuarios y configuración quedaron como deseas.
